\## Descripción

Sistema web desarrollado para registrar, consultar y visualizar reportes relacionados con problemáticas ambientales en el contexto local, como acumulación de residuos, contaminación y quemas no controladas.



\## Objetivo

Diseñar y desarrollar un prototipo funcional basado en arquitectura cliente-servidor que permita organizar información ambiental y apoyar su seguimiento.



\## Tecnologías utilizadas

\- HTML5

\- CSS3

\- Bootstrap 5

\- JavaScript

\- PHP

\- MySQL

\- Leaflet (mapas)



\## Funcionalidades

\- Registro de reportes ambientales

\- Consulta de reportes

\- Carga opcional de imágenes

\- Dashboard estadístico

\- Geolocalización básica

\- Visualización en mapa



\## Metodología

CDIO (Concebir, Diseñar, Implementar y Operar)
